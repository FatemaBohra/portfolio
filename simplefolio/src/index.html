Face Detect - smartBrain

Face Detect - smartBrain is a web application that utilizes facial recognition technology to detect faces in images. It allows users to register an account, sign in, and submit an image URL for face detection. The application keeps track of the number of images the user has submitted and displays the count on their profile.
The project consists of a frontend and a backend. The frontend is built using React, while the backend is implemented with Node.js and Express. The communication between the frontend and backend is done through API calls.
The project showcases the integration of frontend and backend technologies, as well as the usage of external APIs for advanced functionality. It demonstrates the implementation of user authentication, data storage using a PostgreSQL database, and the handling of API requests for image processing.
